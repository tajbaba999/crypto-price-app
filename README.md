## Tech Stack

**Client:** React, ChakraUI

**Server:** Node, Express, Vercel


## Run Locally

Clone the project

```bash
git clone https://github.com/tajbaba999/crypto-price-app.git
```

Go to the project directory

```bash
cd crypto-price-app
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```
